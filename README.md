# Snake Game

## Overview

Snake Game is a browser-based recreation of the classic arcade game built using HTML, CSS, and JavaScript. The objective is simple: control the snake, eat the food to grow longer, and avoid colliding with the walls or your own body. As the snake grows, the game becomes increasingly challenging, making it a fun way to test your reflexes and coordination.

This project was developed to strengthen core web development skills, including DOM manipulation, event handling, game logic, collision detection, and JavaScript programming.

---

## Features

- Classic Snake gameplay
- Smooth keyboard controls
- Random food generation
- Real-time score tracking
- Snake grows after eating food
- Collision detection with walls and the snake's body
- Game Over screen
- Responsive and clean user interface
- Lightweight and runs entirely in the browser

1. Clone this repository:


## Game Rules

- Guide the snake using the arrow keys.
- Eat the food to increase your score.
- Every food item increases the snake's length.
- Avoid hitting the walls.
- Avoid colliding with the snake's own body.
- The game ends immediately upon collision.
